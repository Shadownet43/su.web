Pertanyaan Populer

Jawaban singkat untuk pertanyaan umum.

Apakah saya terlalu muda untuk bergabung dengan komunitas?

+

Tidak, selama Anda memiliki perangkat yang terhubung ke internet, Anda dapat bergabung dengan kami.

Saya tidak punya modal untuk berinvestasi, bisakah saya menghasilkan uang?

+

Ada 19 cara menghasilkan uang, 10 di antaranya tidak memerlukan investasi uang awal untuk memulai, seperti kerja lepas dan copywriting.

Bisakah saya membatalkan langganan saya?

+

Anda dapat membatalkan langganan kapan saja jika Anda merasa platform tersebut tidak cocok untuk Anda.

Seberapa cepat saya dapat menghasilkan uang setelah bergabung?

+

Jika Anda berkomitmen dan sudah memiliki beberapa pengalaman, Anda dapat mulai menghasilkan uang pada minggu pertama.

Saya tidak tinggal di Bandung, Indonesia, apakah ini masalah?

+

Spiritual University tidak hanya untuk penduduk Indonesia, kami adalah platform global yang memiliki mahasiswa dari Malaysia, Brunei, Saudi Arabia dan banyak provinsi di Indonesia.

Saya hanya punya 1 jam sehari, apakah cukup untuk menghasilkan uang?

+

Ya, selama Anda fokus dan memprioritaskan kegiatan-kegiatan yang berpotensi memberikan hasil tinggi yang akan diajarkan dalam kursus.

Apakah ini situs web Ceng Jauhari?

+

Spiritual University adalah situs web resmi Ceng Jauhari. Anda dapat mendaftar dan bergabung dengan komunitas dengan aman melalui situs web ini.
